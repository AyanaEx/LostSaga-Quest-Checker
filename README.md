Quest Checker for bot discord, Use nodejs
=============

#### First you add .env files

```env
DB_USER=YOUR_USR
DB_PASSWORD=YOUR_PWD
DB_SERVER=172.0.0.1
DB_PORT=1433
DB_NAME=LosaGame
DISCORD_BOT_TOKEN=YOUR_DISCORD_BOT_TOKEN
ALLOWED_USERS=YOUR_DISCORD_ID
```
### Simple Run
- Open CMD
- Install require
- NPM i dotenv, discord.js, mssql
- Node losa.js
